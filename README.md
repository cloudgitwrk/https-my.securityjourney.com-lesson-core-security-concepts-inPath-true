# James Chamberlain
    May 28, 2024
    On GEMU, you just create repos the same as you would in GHES. To assign a team a role on the repo, you just go into the repo settings and add the team with the appropriate role.
    We also have a migration tool to facilitate migrating your repos from GHES to GEMU. You can request access to it by using our TaaS Self-Service Portal (TSSP) and choosing the GEMU app and migration as the request type.
